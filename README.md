# Boltverse Official Container

Desktop / Android / iOS web shell for the **Boltverse Games Section** (digital Thunderwolf Citadel).

---

## Play now (no install, no local host)

### **[Open Boltverse →](https://starboltsprint.github.io/boltverse-container/)**

**Direct HTML link:**  
https://starboltsprint.github.io/boltverse-container/

Click that link in any browser. Choose **Desktop**, **Android**, or **iOS**, then enter the Citadel.

---

## What’s in v0.1

- **Platform chooser first**: Desktop / Android / iOS (saved in `localStorage`)
- **Desktop shell**: side nav + wide Home layout
- **Android / iOS shell**: phone frame + bottom tabs + compact chrome
- **Switch layout** anytime from the top bar
- **Home**: Core hero, quick actions, featured, continue, dailies, event banner
- **Discover / Create / Pack / Codex**: scaffold + mock data

Hosted on **GitHub Pages** — public, free, no local server.

---

## Run locally (optional)

```bash
npm install
npm run dev
```

Open http://localhost:5173

```bash
npm run build
npm run preview
```

## Stack

- Vite 6 · React 19 · React Router 7 (HashRouter) · TypeScript
- Auto-deploy to GitHub Pages on every push to `main`
